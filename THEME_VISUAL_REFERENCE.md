## 🔄 Theme Toggle Flow

```
User opens Settings
    │
    ├─► Toggle "Dark Mode" ON
    │       │
    │       ├─► colorScheme detects change
    │       │
    │       ├─► isDarkMode = true
    │       │
    │       └─► themedColors returns dark palette
    │
    ├─► Navigate to Outfit Scorer
    │       │
    │       └─► All components use dark colors ✓
    │
    ├─► Navigate to AI Stylist
    │       │
    │       └─► All components use dark colors ✓
    │
    └─► Toggle "Dark Mode" OFF
            │
            ├─► colorScheme detects change
            │
            ├─► isDarkMode = false
            │
            └─► themedColors returns light palette

All components update immediately ✨
```

## 🚦 Quick Test Script

1. **Open app** → Should be in light mode (default)
2. **Go to Outfit Scorer** → All white/light colors ✓
3. **Go to Settings** → Toggle Dark Mode ON
4. **Return to Outfit Scorer** → All dark colors ✓
5. **Take/select photo** → Image container dark ✓
6. **Fill context field** → Dark input with light text ✓
7. **Analyze outfit** → Score cards dark ✓
8. **Go to AI Stylist** → Permission screen dark ✓
9. **Grant permission** → Message bubbles dark ✓
10. **Go to Settings** → Toggle Dark Mode OFF
11. **Return to Outfit Scorer** → All light colors ✓
12. **Return to AI Stylist** → All light colors ✓

✅ **PASS:** No component stuck in wrong theme!

## 📱 Platform-Specific Notes

### iOS:

- System dark mode: Settings → Display & Brightness → Dark
- App respects both system and in-app toggle

### Android:

- System dark mode: Settings → Display → Dark theme
- App respects both system and in-app toggle

### Web:

- Browser dark mode varies by browser
- App respects in-app toggle primarily
